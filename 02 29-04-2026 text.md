# 🐧 Linux Practice Lab – Beginner to Intermediate

## 📌 1. System Preparation & Package Management

### 🔹 Clear Screen

```bash
clear
```

### 🔹 Update System Packages

```bash
sudo yum update -y
```

### 🔹 Check Python Version

```bash
python --version
```

### 🔹 Remove & Install Python

```bash
sudo yum remove python -y
sudo yum install python -y
```

---

## 📌 2. Install Utilities

### 🔹 Install `tree` (Directory Visualization)

```bash
tree --version
sudo yum install tree -y
tree --version
```

### 🔹 Install `nano` Editor

```bash
sudo yum install nano -y
```

---

## 📌 3. Basic File & Directory Operations

### 🔹 Create Directories & Files

```bash
mkdir project
touch a.txt
mkdir .myproject      # hidden directory
touch .b.txt          # hidden file
```

### 🔹 List Files

```bash
ls
ls -la   # shows hidden files
```

---

## 📌 4. Directory Structure Visualization

```bash
tree
tree -a   # include hidden files
```

---

## 📌 5. Move & Organize Files

```bash
mv a.txt project
mkdir new
mv new project
```

---

## 📌 6. Navigation Commands

```bash
cd project
pwd
cd ..
cd /
cd home
cd ec2-user/
cd project/
```

---

## 📌 7. File Editing

### 🔹 Using Nano

```bash
nano a.txt
nano b.txt
```

### 🔹 Using Vi/Vim

```bash
vi --version
vi b.txt
```

---

## 📌 8. File Viewing Commands

```bash
cat a.txt
less b.txt
more b.txt
head b.txt
tail b.txt
```

---

## 📌 9. Copy & Move Files

```bash
cp a.txt b.txt        # copy file
mv a.txt new          # move file
```

### 🔹 Copy Directory

```bash
cp -r new hello
```

---

## 📌 10. Create Additional Structure

```bash
mkdir hello
cp -r new hello
tree
```

---

## 📌 11. System Information Commands

```bash
uname        # OS info
uname -r     # kernel version
whoami       # current user
```

❌ Incorrect commands used:

```bash
whoiam   # wrong
whoim    # wrong
```

---

## 📌 12. Output Redirection

```bash
echo "hello world"
echo "hello world" >> a.txt   # append
echo "hello" > a.txt          # overwrite
cat a.txt
```

---

## 📌 13. Permissions & Root Access (Important Concept)

```bash
sudo mkdir new   # create directory with root permission
```

👉 Use `sudo` when:

* Permission denied
* System-level changes

---

## 📌 14. History Tracking

```bash
history
```

## 📌 15. Common Mistakes (From Your Practice)

| Mistake        | Correct Command   |
| -------------- | ----------------- |
| `clrea`        | `clear`           |
| `whoiam`       | `whoami`          |
| `whoim`        | `whoami`          |
| `cp new hello` | `cp -r new hello` |

# 🎯 Final Learning Flow (IMPORTANT)

Follow this order when practicing:

1. System setup (`yum`, install tools)
2. File creation (`mkdir`, `touch`)
3. Listing (`ls`, `ls -la`)
4. Structure (`tree`)
5. Navigation (`cd`, `pwd`)
6. Editing (`nano`, `vi`)
7. Viewing (`cat`, `less`, `more`)
8. Copy/Move (`cp`, `mv`)
9. Redirection (`echo > >>`)
10. System info (`uname`, `whoami`)
11. Permissions (`sudo`)
12. History tracking

---

# 🚀 Pro Tips (Exam + Real World)

* Always use `ls -la` → shows hidden files (very important)
* Use `tree -a` → better visualization
* `>` overwrites, `>>` appends (frequently asked)
* `cp -r` → mandatory for directories
* `sudo` = admin access (critical in cloud/EC2)
