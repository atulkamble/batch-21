### ✅ Steps

### 1. Enable WSL

```powershell
wsl --install
```

---

### 2. Download RHEL Root Filesystem

Go to:
👉 [https://developers.redhat.com/products/rhel/download](https://developers.redhat.com/products/rhel/download)

Download **RHEL Binary (tar.gz / rootfs)**

---

### 3. Import into WSL

```powershell
mkdir C:\WSL\RHEL

wsl --import RHEL C:\WSL\RHEL rhel-rootfs.tar.gz
```

---

### 4. Run RHEL

```powershell
wsl -d RHEL
```

---

### 5. Set User (optional)

```bash
useradd -m atul
passwd atul
```

# ⚠️ Important Notes

* RHEL requires **subscription activation**

```bash
subscription-manager register
```

* Some features may not work in WSL:

  * ❌ systemd (older WSL versions)
  * ❌ kernel-level features
  * ❌ full virtualization
